# Bleumr

AI-powered desktop browser and agent platform.

## Download

Get the latest release from the [Releases page](https://github.com/authenticrisktakers-glitch/Bleumr.app/releases).

### macOS Installation

After downloading the `.dmg`:
1. Open the DMG and drag Bleumr to Applications
2. If macOS blocks it, run: `xattr -cr /Applications/Bleumr.app`
3. Then open normally
