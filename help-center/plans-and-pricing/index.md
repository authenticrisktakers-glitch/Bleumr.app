# Plans and Pricing

## Overview

Bleumr offers flexible subscription tiers designed to fit different usage levels. All tiers include access to the core chat interface, web search, and basic tools. Premium tiers unlock advanced AI models, higher usage limits, and specialized features.

## Plan Comparison

### Free
**Cost:** $0/month

- 15 AI calls per day
- Access to Llama 3.1-8B model
- Chat with web search
- Image analysis (upload and discuss images)
- Code Lab (write, edit, and run code)
- Web Designer (AI website builder)
- Calendar and Scheduler
- Basic Orbits (background task monitoring)
- Community support

### Pro
**Cost:** See bleumr.com for current pricing

- 150 AI calls per day
- Access to Llama 3.3-70B model (more capable, nuanced responses)
- All Free features, plus:
- Full browser automation
- Trading Dashboard (crypto and stock tracking)
- BLEU BASE GG (AI game world generator)
- Workspace (multi-agent orchestration)
- Voice Chat
- Full Orbits with unlimited active goals
- Email support

### Stellur (Founder Tier)
**Cost:** By invitation or founder license

- Unlimited AI calls
- Access to all current and future models
- All Pro features, plus:
- Priority support
- Early access to new features
- Founding member recognition

## How Usage Limits Work

- Each AI interaction (sending a message and receiving a response) counts as one call.
- Web searches triggered within a conversation count as part of the same call.
- Image analysis counts as one call per image analyzed.
- Usage resets daily at midnight UTC.
- You can monitor your remaining daily calls in **Settings > Plan**.

## Upgrading Your Plan

1. Visit bleumr.com or contact us to purchase a Pro or Stellur license key.
2. Open Bleumr and go to **Settings > Plan**.
3. Enter your license key and click **Activate**.
4. Your new tier takes effect immediately.

## Using Your Own API Key

If you prefer to use your own Groq API key:
1. Go to **Settings > Engine**.
2. Enter your Groq API key.
3. When using your own key, Bleumr does not count calls against your subscription limit. Your usage is governed by your Groq account's rate limits and billing.

## Fair Use

All plans are subject to our fair use policy. Automated scripting, bulk data extraction, or any use that places disproportionate load on our infrastructure may result in temporary throttling or account review. If you have high-volume needs, contact us to discuss enterprise options.
