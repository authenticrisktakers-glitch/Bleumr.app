# BLEU BASE GG (AI Game World Generator)

> **Availability:** Desktop app only. Pro and Stellur tiers only.

## Overview

BLEU BASE GG is an AI-powered 2D game world generator. Upload a piece of concept art, a sketch, or a photo — or simply describe a world in words — and BLEU BASE GG transforms it into a living, explorable 2D side-scrolling environment. Every frame is generated by AI in real time as you move through the world.

## How It Works

1. Open **Apps** from the sidebar and click **BLEU BASE GG**.
2. Choose your starting point:
   - **Upload an image**: Drag and drop concept art, a sketch, or a photo into the upload zone. Add a text description to help the AI understand your image.
   - **Describe a world**: Type a description of the world you want to explore.
   - **Pick a preset**: Choose from 8 pre-built world templates.
3. BLEU BASE GG analyzes your input and generates the first frame of your world.
4. The world auto-explores, generating new AI frames as it moves through the scene.
5. Use arrow keys, WASD, or the on-screen buttons to steer the exploration direction.

## Features

### Auto-Explore Mode
When enabled (default), BLEU BASE GG automatically advances through the world, generating new frames continuously without user input. The exploration mostly moves forward (right) with occasional direction changes for variety.

- Toggle auto-explore with the Play/Pause button.
- While auto-exploring, you can still press direction keys to steer.

### Frame Caching
Visited locations are cached in memory. If you return to a previously visited position, the frame loads instantly without regeneration.

### Dreamy Visual Style
Frames are rendered at a lower resolution with a dreamy visual treatment:
- Soft blur and enhanced saturation for a painterly feel.
- Vignette and film grain overlays.
- Glowing border around the viewport.

### World Consistency
The AI maintains consistency across frames by using:
- A fixed art style, color palette, and terrain type for all frames in a world.
- Position-derived seeds so the same location always generates the same visual.
- Landmark references that connect frames spatially.

## How Frame Generation Works

Each frame is generated through a two-step process:
1. **Prompt Construction**: Based on the world description, current position, and movement direction, a detailed image prompt is constructed locally.
2. **Image Generation**: The prompt is sent to an AI image generation model that produces the frame.

Frame generation typically takes 2-5 seconds depending on server load. During generation, a "rendering..." indicator appears.

## Limitations

- Frame generation requires a stable internet connection and depends on the availability of the image generation service.
- Visual consistency between frames is approximate. Since each frame is independently generated, there may be variations in details, colors, or object placement between adjacent frames.
- The world is not truly interactive in the way a traditional video game is. There are no physics, collision detection, or game mechanics. It is a visual exploration experience.
- Frame generation speed depends on server load and may vary.
- Complex or highly specific scene descriptions may not always be accurately rendered.
