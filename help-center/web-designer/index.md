# Web Designer

## Overview

Web Designer is Bleumr's AI-powered website builder. Describe the website you want in plain language, and JUMARI generates a complete, working website with HTML, CSS, and JavaScript. Edit the design through conversation — tell JUMARI what to change, and it updates the code and preview in real time.

## How It Works

1. Open **Apps** from the sidebar and click **Web Designer**.
2. Describe the website you want. For example: "Create a modern portfolio website for a photographer with a dark theme, image gallery, and contact form."
3. JUMARI generates the website code and displays a live preview side by side.
4. Request changes through the chat. For example: "Make the header sticky," "Change the accent color to blue," or "Add a testimonials section."
5. Export your finished project as a ZIP file.

## Features

### Live Preview
- See your website rendered in real time as JUMARI generates and modifies the code.
- The preview updates automatically with each change.
- Toggle between desktop, tablet, and mobile viewport sizes to test responsive design.

### Virtual File System
- Each project maintains a virtual file system with separate HTML, CSS, and JavaScript files.
- Browse files in the file tree panel.
- View and edit source code directly in the code viewer.

### Project Management
- **Create**: Start a new project from scratch or from a description.
- **Save**: Projects are saved locally and persist between sessions.
- **Delete**: Remove projects you no longer need.
- **Export**: Download your project as a ZIP file containing all HTML, CSS, and JavaScript files.

### Console Panel
- View browser console output (logs, errors, warnings) from your website's JavaScript.
- Debug runtime errors with real-time feedback.

### Auto-Debug
When JUMARI detects a runtime error in your website's JavaScript, it automatically suggests a fix.

### Code Quality Audit
Request a code quality audit to receive feedback on:
- Accessibility (WCAG compliance)
- Performance optimization
- SEO best practices
- Code structure and maintainability

### Design Templates
Choose from a library of starting templates to accelerate your project.

## Editing Through Conversation

Web Designer uses a conversational interface. Instead of dragging and dropping elements, you describe changes in natural language:

- "Add a hero section with a background video."
- "Make the navigation responsive with a hamburger menu on mobile."
- "Change all headings to use the Poppins font."
- "Add smooth scroll animation when clicking navigation links."
- "Create a pricing table with three tiers."

JUMARI interprets your request, modifies the relevant files, and updates the preview.

## Exporting Your Website

1. Click the **Export** button in the Web Designer toolbar.
2. A ZIP file containing your project files will be downloaded.
3. Upload these files to any web hosting provider (Netlify, Vercel, GitHub Pages, shared hosting, etc.) to publish your website.

> **Note:** Exported websites are static HTML/CSS/JS files. They do not require Bleumr or any special server to run. Any standard web server can host them.

## Limitations

- Web Designer generates static websites (HTML, CSS, JavaScript). It does not generate backend code, databases, or server-side logic.
- Complex web applications with state management, authentication, or database integration are beyond the current scope of Web Designer.
- The quality and complexity of generated websites depend on the clarity of your description and the capabilities of the underlying AI model.
- Live preview runs in a sandboxed iframe. Some browser features may behave differently in the sandbox compared to a standalone browser.
- AI-generated code may contain errors or suboptimal patterns. Review the code before publishing.
