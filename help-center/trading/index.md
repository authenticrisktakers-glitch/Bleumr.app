# Trading Dashboard

> **Availability:** Desktop app only. Pro and Stellur tiers only.

## Overview

The Trading Dashboard provides live cryptocurrency and stock price tracking, portfolio monitoring, and price alerts. It connects to major exchanges to display real-time market data and allows you to manage a watchlist of assets you care about.

## Features

### Live Price Feeds
- Real-time price data for cryptocurrencies and stocks.
- Prices update automatically at regular intervals.
- Display includes current price, 24-hour change (amount and percentage), and price direction indicators.

### Supported Exchanges
- Binance
- Coinbase
- Kraken

### Default Watchlist
The Trading Dashboard comes with a default watchlist including popular assets such as BTC, ETH, SPY, and others. You can customize your watchlist by adding or removing assets.

### Portfolio Tracking
- Add your holdings to track your portfolio's total value.
- View gains and losses in real time.
- Portfolio calculations are performed locally — your holdings data is not transmitted to any server.

### Price Alerts
- Set price alerts for any asset on your watchlist.
- Configure threshold values (e.g., "Alert me when BTC drops below $50,000").
- Receive in-app notifications when an alert triggers.

### Trade History
- View a log of your tracked trades.
- Trades are recorded locally for your reference.

### Exchange Credentials
- Optionally connect exchange API keys for enhanced data access.
- API keys are stored securely using Bleumr's secure storage system.

## Using the Trading Dashboard

1. Open **Apps** from the sidebar.
2. Click **Trading**.
3. Browse your watchlist and portfolio.
4. Click an asset for detailed price information.
5. Use the alert button to set price notifications.
