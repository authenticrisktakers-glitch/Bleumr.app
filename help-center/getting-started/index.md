# Getting Started with Bleumr

## What is Bleumr?

Bleumr (JUMARI 1.0) is an AI-powered desktop platform that combines conversational AI, browser automation, code development tools, and creative applications into a single unified experience. Bleumr is available as a desktop application for macOS and as a progressive web app (PWA) for supported browsers.

## System Requirements

### Desktop App (Recommended)
- **macOS**: 11.0 (Big Sur) or later
- **RAM**: 8 GB minimum, 16 GB recommended
- **Storage**: 500 MB free disk space
- **Internet**: Broadband connection required for AI features

### Web App (PWA)
- **Browser**: Google Chrome 90+, Microsoft Edge 90+, or Safari 16+
- **Internet**: Broadband connection required

> **Note:** Certain features, including browser automation, the trading dashboard, and BLEU BASE GG, are only available in the desktop application.

## Installation

### Desktop App
1. Download Bleumr from the official website.
2. Open the downloaded `.dmg` file.
3. Drag the Bleumr icon into your Applications folder.
4. Open Bleumr from your Applications folder or Launchpad.
5. If prompted by macOS Gatekeeper, click "Open" to confirm you trust the application.

### Web App
1. Navigate to the Bleumr web app in a supported browser.
2. When prompted, tap "Add to Home Screen" or "Install App."
3. Open Bleumr from your home screen for a native-like experience.

## Initial Setup

### Onboarding
When you first open Bleumr, you will be guided through a brief onboarding flow:

1. **Profile Creation** — Enter your name, email, and basic information. This data is stored locally on your device and is used to personalize your experience.
2. **API Key Configuration** (Optional) — If you have your own Groq API key, you can enter it in Settings to use your own quota. Otherwise, Bleumr provides shared access based on your subscription tier.
3. **Explore the Interface** — After onboarding, you will see the main chat interface with the sidebar navigation.

### Navigation
- **Sidebar**: Access all features including Chat, Apps (Code Lab, Web Designer, Trading, BLEU BASE GG), Orbits, Workspace, Calendar, and Settings.
- **Chat**: The primary interface. Type messages, ask questions, upload images, or give commands.
- **Apps**: Open the Apps launcher from the sidebar to access specialized tools.

## First Steps

1. **Start a conversation** — Type a message in the chat input to interact with JUMARI, the AI assistant.
2. **Try a web search** — Ask JUMARI to search the web for real-time information.
3. **Upload an image** — Drag and drop or click the attachment icon to share an image for AI analysis.
4. **Explore Apps** — Click the Apps button in the sidebar to discover Code Lab, Web Designer, Trading, and BLEU BASE GG.

## Getting Help
